25 /*
26  * This covers the "version 2" RTC peripheral. This is completely different
27  * to the v1 RTC periph on the F1 series devices. It has BCD counters, with
28  * automatic leapyear corrections and daylight savings support.
29  * This peripheral is used on the F0, F2, F3, F4 and L1 devices, though some
30  * only support a subset.
31  */
