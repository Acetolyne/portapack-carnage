109 /*
110  * I/O ports initial setup, this configuration is established soon after reset
111  * in the initialization code.
112  *
113  * The digits have the following meaning:
114  * 0 - Analog input.
115  * 1 - Push Pull output 10MHz.
116  * 2 - Push Pull output 2MHz.
117  * 3 - Push Pull output 50MHz.
118  * 4 - Digital input.
119  * 5 - Open Drain output 10MHz.
120  * 6 - Open Drain output 2MHz.
121  * 7 - Open Drain output 50MHz.
122  * 8 - Digital input with PullUp or PullDown resistor depending on ODR.
123  * 9 - Alternate Push Pull output 10MHz.
124  * A - Alternate Push Pull output 2MHz.
125  * B - Alternate Push Pull output 50MHz.
126  * C - Reserved.
127  * D - Alternate Open Drain output 10MHz.
128  * E - Alternate Open Drain output 2MHz.
129  * F - Alternate Open Drain output 50MHz.
130  * Please refer to the STM32 Reference Manual for details.
131  */
132

134  * Port A setup.
