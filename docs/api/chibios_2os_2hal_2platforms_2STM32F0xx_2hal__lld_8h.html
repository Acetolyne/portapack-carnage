Detailed Description

STM32F0xx HAL subsystem low level driver header.

Precondition
This module requires the following macros to be defined in the board.h file:
  • STM32_LSECLK.
  • STM32_LSEDRV.
  • STM32_LSE_BYPASS (optionally).
  • STM32_HSECLK.
  • STM32_HSE_BYPASS (optionally).
One of the following macros must also be defined:
  • STM32F030 for Value Line devices.
  • STM32F0XX_LD for Low Density devices.
  • STM32F0XX_MD for Medium Density devices.
